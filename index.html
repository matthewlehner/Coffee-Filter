Coffee Filter

Coffeescript helpers for more...drinkable projects

Download .zip Download .tar.gz View on GitHub

Coffee Filter is a collection of coffeescript helpers I've used across projects. I didn't make all of them and some I customized, but I've credited other authors where appropriate.

Special thanks to Dtime, the company at which I wrote most of these.

Included are (click to view example):

  • hover tags for showing notes on an element (e.g., a username when hovering over an avatar)
  • placeholder text for browsers that don't support the html5 spec (requires modernizr)
  • textarea autoexpand when typing large blocks of text
  • fill a square with an image of arbitrary dimensions
  • position an image of arbitrary dimensions in a square
  • saving and default states for buttons/links when using ajax
  • autolink urls in a block of text (great for comments!)
  • set a custom tabindex on an array of items
  • outerHTML()
  • a check to see if keyboard shortcuts should be enabled (if not ie8 or typing in a field)

Placeholder Text

Just add your placeholder text as you would in html5. 

<input placeholder='placeholder text' type='text' />

To see that this works, you'll have to run a browser that doesn't support placeholder text (like IE9 or below).

Autoexpanding Text Areas

Call .autoexpand() on the textarea of your choice! Optionally, pass a function in to get run every time the textarea grows or shrinks. 

$('#autoexpand textarea').autoexpand(autoexpand_notifer)

Saving State for Ajax Forms

Saving a form via ajax? Want to give the user some feedback? Click the button to the left to see how it looks.

It works by setting data attributes on your link or button:
  • data-original-text: what the button or link will display
  • data-saving-text: the text to display while saving
  • data-saved-text: optional, if present it shows for 1 second before reverting to the original text
    • <a href='#' class='fancy_button' data-original-text='click to save' data-saving-text='saving...' data-saved-text='saved!'></a>

    Note that there's no text in the anchor tag; it's only in the attribute. So on load, run this function, which goes through the buttons that have the original-text attribute and sets the text: 
    window.set_original_text()

    Then you can call this function on click: 
    $('.fancy_button').click(function(){
  $(this).save_state();
})

    And to get it back to normal, just run this function, maybe in the success callback of your ajax request: 
    $('.fancy_button').unsave_state();

    Consider adding styles to the .saving class on your button to remove any hover or click states that you might have on your button.

Autolink Text

You can see my portfolio at www.mgeraci.com or http://www.michaelgeraci.com, check it out!
Just run .link_urls() on any block of text that you want linked: 
$('#to_autolink').link_urls()

It will only run once on a given element, so you can run it on a class and later append other elements of that class without it trying to re-link existing elements.

The regex used does a pretty good job, but only catches urls that start with http:// or www.

Custom Tabindex




Tap tab to see the custom tabindex in action. Useful for helping users navigate comple forms with the keyboard.

html: 
<input type='text' class='tabindex4' />
<input type='text' class='tabindex3' />
<input type='text' class='tabindex2' />
<input type='text' class='tabindex1' />

In javascript, call pass an array of jQuery objects in the order that you want tabbed: 
window.set_tabindex([
  $('.tabindex1'),
  $('.tabindex2'),
  $('.tabindex3'),
  $('.tabindex4')
])

And here's .outerHTML(): 
$('#to_outerhtml').html()

Keyboard Shortcut Checker

Keyboard shortcuts are cool! Until you're typing a form and trigger them by accident. In the code for your keyboard shortcuts, add a this check: 
window.should_allow_keyboard_shortcuts()
